# @arvo-tools/agentic

Agentic toolset for building applications using arvo-core and arvo-event-handler.

## Installation

\`\`\`bash
npm install @arvo-tools/agentic
# or
pnpm add @arvo-tools/agentic
\`\`\`

## Usage

\`\`\`typescript
import { version } from '@arvo-tools/agentic';

console.log(version);
\`\`\`
